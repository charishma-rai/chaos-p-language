# P Language Translator

This is a small website that converts normal text into **P-language**.

If you type something like:

hello world

you’ll get something like:

hepellopo woporld

Basically it adds a little pattern to vowels and turns your sentence into a funny secret language.

---

## What it can do

- Live translation while typing  
- Chaos mode (makes the translation even more ridiculous)  
- Copy button so you can send the result to someone  

---

## Try it

Type random words and see what happens.  
You might start noticing the pattern after a while.

There are also a few hints on the page if you're curious about how it works.

---

## Built with

Just simple stuff:

- HTML  
- CSS  
- JavaScript  

No frameworks, nothing fancy.

---

## Why this exists
Mostly for fun.
Secret languages are interesting, and it's fun seeing how a tiny rule can completely change how words look.
